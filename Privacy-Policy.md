# Privacy Policy

**Effective Date:** April 26, 2026

This Privacy Policy explains how **RusType** (“the Extension”, “we”, “our”, or “us”) handles user data.

The Extension is designed to help Vietnamese users write Russian text more accurately by providing spelling, grammar, autocomplete, and writing suggestions directly in editable fields on supported websites.

By using the Extension, you agree to the practices described in this Privacy Policy.

---

## 1. Single Purpose of the Extension

The single purpose of the Extension is to provide Russian writing assistance for Vietnamese users.

The Extension may help users with:

- Russian spelling correction;
- grammar suggestions;
- autocomplete suggestions;
- writing improvement suggestions;
- explanations of writing issues in Vietnamese;
- personal dictionary features;
- enabling or disabling the Extension on supported websites.

The Extension does not provide unrelated services and does not use user data for purposes unrelated to Russian writing assistance.

---

## 2. Data the Extension May Process

To provide its core writing assistance features, the Extension may process text that the user enters, selects, or edits in editable fields on supported websites.

This may include:

- text typed into input fields;
- text typed into text areas;
- text typed into contenteditable fields;
- selected text that the user asks the Extension to check;
- text inside supported online editors, such as document editors or email editors.

This type of data may be considered **web content** because it may include text from webpages or editable areas on webpages.

The Extension processes this text only to provide spelling, grammar, autocomplete, and writing suggestions.

---

## 3. Data Stored Locally

The Extension may store some information locally in the user’s browser using Chrome storage.

This may include:

- extension settings;
- enabled or disabled status;
- language preferences;
- writing preferences;
- personal dictionary entries;
- dismissed suggestions;
- website-specific preferences.

This data is stored locally to make the Extension work consistently across browsing sessions.

The Extension does not use locally stored data for advertising, profiling, credit scoring, or unrelated purposes.

---

## 4. Data Not Collected

The Extension does not intentionally collect the following categories of personal or sensitive user data:

- personally identifiable information, such as full name, address, identification number, or date of birth;
- health information;
- payment or financial information;
- authentication information, such as passwords, login credentials, security questions, or PINs;
- personal communications unrelated to writing assistance;
- precise location information;
- browsing history;
- credit information;
- biometric information.

Users should avoid submitting passwords, payment details, identification documents, or other sensitive information for checking through the Extension.

---

## 5. Use of Data

The Extension uses processed text and local settings only for the following purposes:

- to detect spelling errors;
- to detect possible grammar issues;
- to generate autocomplete suggestions;
- to provide writing improvement suggestions;
- to explain writing issues to the user;
- to save user preferences;
- to maintain the user’s personal dictionary;
- to improve the functionality and reliability of the Extension.

The Extension does not use user data for advertising, user profiling, resale, credit scoring, or unrelated analytics.

---

## 6. Third-Party Services

To provide spelling, grammar, or writing suggestions, the Extension may send text entered or selected by the user to external language-processing services.

These services may include spelling-checking APIs, grammar-checking APIs, autocomplete services, or AI-based language-processing services.

The text is sent only when necessary to provide the Extension’s core writing assistance features.

Third-party services are not permitted by us to use the submitted text for advertising, credit scoring, or unrelated purposes.

However, users should understand that data sent to third-party services may be processed according to the privacy policies and terms of those services.

---

## 7. Remote Code

The Extension does not use remote code.

The Extension does not download or execute JavaScript, WebAssembly, or other executable code from a remote server.

The Extension may communicate with external APIs to request spelling, grammar, autocomplete, or writing suggestions. Such API communication is used only to provide the Extension’s core features and is not considered remote code execution.

---

## 8. Data Sharing

The Extension does not sell user data.

The Extension does not transfer user data to third parties except when necessary to provide its core writing assistance features, such as sending user-selected or user-entered text to a language-processing API.

The Extension does not share user data with advertisers, data brokers, credit agencies, or unrelated third parties.

---

## 9. Data Retention

Text processed for spelling, grammar, autocomplete, or writing suggestions is used only for the current request.

The Extension does not intentionally store checked text permanently on its own servers.

Locally stored settings and personal dictionary entries remain in the user’s browser until the user changes them, clears browser data, disables storage, or uninstalls the Extension.

---

## 10. User Control

Users can control the Extension by:

- enabling or disabling the Extension;
- disabling it on specific websites, if supported;
- clearing local browser storage;
- deleting personal dictionary entries, if supported;
- uninstalling the Extension from the browser.

If the user uninstalls the Extension, locally stored Extension data may be removed by the browser according to Chrome’s extension storage behavior.

---

## 11. Security

We take reasonable steps to limit data processing to what is necessary for the Extension’s core purpose.

The Extension is designed to request only the permissions needed to provide writing assistance.

However, no method of electronic transmission or storage is completely secure, and we cannot guarantee absolute security.

---

## 12. Children’s Privacy

The Extension is not specifically directed to children under the age required by applicable law.

The Extension does not knowingly collect personal information from children.

If you believe that a child has provided personal information through the Extension, please contact us so we can take appropriate action.

When we update this Privacy Policy, we will change the effective date at the top of this page.

Continued use of the Extension after changes means that the user accepts the updated Privacy Policy.
